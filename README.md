# URL Shortener

A simple URL Shortener built with **Node.js**, **Express**,**React.js** and **MongoDB**. It allows users to convert long URLs into short, easy-to-share links.

## Features

- Shorten long URLs
- Generate unique short codes
- Redirect to the original URL
- Store URLs in MongoDB
- Custom short code support 

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
